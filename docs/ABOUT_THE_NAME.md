# About the name — this project is renaming to **GhostDeck**

## This is an independent, unofficial project
This application is **not made by, affiliated with, endorsed, sponsored or supported by MSI**
(Micro-Star International Co., Ltd.). It is a free, open-source community tool that talks to the
laptop's embedded controller through MSI's own public WMI interface to restore the **Silent**
performance profile and add a few conveniences (fan control, charge limit, a gaming overlay).

"MSI", "MSI Center" and "Cooler Boost" are trademarks of Micro-Star International. They are used
here **only descriptively**, to state which hardware the tool is compatible with.

## Why rename?
To make that independence unmistakable and to avoid using MSI's trademarks in our own product name,
the app is moving from **"MSI Profile Switcher"** to:

> ## GhostDeck — for MSI laptops

The word "MSI" stays only as a plain compatibility descriptor ("for MSI laptops"), never as our brand.
The in-app **"Cooler Boost"** feature was already renamed to the generic **"Fan Boost"** for the same
reason (it remains equivalent to MSI's Cooler Boost).

## What changes for you — nothing to do
- **Same app, same features**, just a new name and icon.
- **Your settings and history are migrated automatically** on first launch of the renamed build
  (profiles, hotkeys, overlay layout, charge limit, change log — all preserved).
- The download will be `GhostDeck.exe` instead of `MSIProfileSwitcher.exe`; after updating you can
  delete the old file. Autostart is re-registered automatically under the new name.
- Update checks and this notice channel are unchanged (read-only, no data collected; you can turn
  them off in Settings).

## Questions or feedback?
Use **Send feedback…** in the tray menu (opens GitHub Discussions), or open the project page.
Thanks for using it — and for helping verify models.
